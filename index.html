Mathematics Solutions:

1. Correct answer: A) 0 - Using remainder theorem, f(2) = (2)³ - 6(2)² + 11(2) - 6 = 8 - 24 + 22 - 6 = 0

2. Correct answer: B) 55° - Sum of angles in triangle = 180°. Right angle = 90°, so 90° + 35° + x = 180° → x = 55°

3. Correct answer: A) 16 cm - Area of rhombus = (d₁ × d₂)/2 → 96 = (12 × d₂)/2 → d₂ = 16 cm

4. Correct answer: A) 4/5 - Using identity sin²θ + cos²θ = 1, cosθ = √(1 - (3/5)²) = √(1 - 9/25) = √(16/25) = 4/5

5. Correct answer: A) (3, 4) - Using section formula, x = (1×6 + 3×2)/(1+3) = (6+6)/4 = 12/4 = 3, y = (1×7 + 3×3)/(1+3) = (7+9)/4 = 16/4 = 4

6. Correct answer: A) 6 - Sₙ = 3n² + 2n, S₁ = 3(1)² + 2(1) = 5, S₂ = 3(4) + 4 = 16, a₁ = 5, a₂ = S₂ - S₁ = 11, d = a₂ - a₁ = 6

7. Correct answer: C) 16 cm - Volume of cone = (1/3)πr²h → 1232 = (1/3)×(22/7)×7²×h → h = (1232×3×7)/(22×49) = 16 cm

8. Correct answer: A) 5/2 - For quadratic ax²+bx+c, sum of roots = -b/a = 5/2

9. Correct answer: A) 1/9 - Possible outcomes: (3,6), (4,5), (5,4), (6,3) → 4/36 = 1/9

10. Correct answer: A) 1/4 - tan A = 1/√3 → A = 30°, sin A cos A = (1/2)(√3/2) = √3/4

11. Correct answer: A) 10 - For even number of terms, median = average of (n/2)th and (n/2+1)th terms → (12+x)/2 = 11 → x = 10

12. Correct answer: A) 7 cm - Perimeter of semicircle = πr + 2r = r(π+2) = 36 → r(22/7+2) = 36 → r(36/7) = 36 → r = 7 cm

14. Correct answer: B) 4:5 - Ratio of areas = (ratio of sides)² → 16:25 = (4:5)²
